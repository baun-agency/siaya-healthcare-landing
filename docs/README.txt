SIAYA HEALTHCARE MEDICAL CENTRE - STATIC WEBSITE
================================================

DEPLOYMENT INSTRUCTIONS FOR GITHUB PAGES:

1. Upload all files from this /docs folder to your GitHub repository

2. Go to your repository Settings > Pages

3. Under "Build and deployment":
   - Source: Deploy from a branch
   - Branch: main (or master)
   - Folder: /docs

4. Click Save and wait a few minutes

5. Your site will be live at:
   https://YOUR-USERNAME.github.io/siaya-healthcare-landing/

ALTERNATIVE: ROOT DEPLOYMENT
----------------------------
If you prefer to deploy from the root:
1. Copy all contents of /docs to the repository root
2. Set GitHub Pages source to / (root) instead of /docs

NOTES:
- All assets are optimized and included
- The 404.html handles client-side routing
- Base path is set to /siaya-healthcare-landing/
